# Diagrams

This directory contains the system diagrams for the Knowledge Graph / Wiki System project.

## Files

- `neo4j_schema.drawio` - Neo4j graph schema diagram (node types, relationships, properties)
- `architecture.drawio` - System architecture diagram (services, ports, data flows)

## Exporting to PNG

To export the diagrams as PNG images:

1. Open the `.drawio` file in [draw.io](https://app.diagrams.net/) (or download the desktop app)
2. Go to **File → Export as → PNG**
3. Choose appropriate resolution (recommended: 300 DPI for high quality)
4. Save as `neo4j_schema.png` or `architecture.png` in this directory

The PNG files are referenced in the main README.md for embedding in documentation.

## Alternative: Online Export

You can also use the draw.io online editor:
1. Visit https://app.diagrams.net/
2. Open the `.drawio` file
3. Export as PNG using the menu
